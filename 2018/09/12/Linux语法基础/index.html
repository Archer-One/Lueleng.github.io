Linux语法基础

Linux语法基本介绍……

Linux

  • Linux为开源软件
  • Linux和Windows区别
    • Linux严格区分大小写；
    • Linux中所有内容以文件形式保存，包括硬件；
    • Linux不靠扩展名区分文件类型(靠权限区分)；
    • Windows下的程序不能直接在Linux中安装和运行。
  • Linux有很多版本，简单介绍字符界面优势
    • 字符界面占用的系统资源更少；
    • 字符界面减少了出错，被攻击的可能性。

Linux基础语法

用户提示符

  • 超级用户提示符

  • $ 普通用户提示符

命令格式

  • 命令 [选项] [参数] （个别命令使用不遵循此格式）

ls 命令

命令 内容
ls -l 显示详细信息
ls -lh 人性化显示详细信息
ls -a 显示所有文件，包括隐藏文件
ls -d 查看目录属性而非目录下的内容
ls -ld 查看目录详细信息
ls -i 显示文件inode

cd、ctrl、mkdir 命令

命令 内容
cd 切换目录
cd ~ = cd + Enter 回到当前用户的家目录
cd - 回到上次所在目录
cd . 进入当前目录
cd .. 回到上级目录
ctrl+c 强制终止当前命令
ctrl+l 清屏
ctrl+a 光标移动到命令行首
ctrl+e 光标移动到命令行尾
ctrl+u 从表所在位置删除到行首
ctrl+z 把命令放入后台
ctrl+r 在历史命令中搜索
mkdir 建立目录，但是不可以递归创建如 mkdir stduy/chinese
mkdir -p 可递归创建目录
rmdir 只能删除空目录

rm -rf 命令

命令 内容
rm -r 删除目录
-f 强制
rm -rf 强制删除
  • rm -rf / 可执行删除根目录命令，相当于令Linux自杀

cp、mv

cp [选项] [原文件或目录] [目标文件]

命令 内容
cp - r 复制目录
cp - p 连带文件属性赋值
cp - d 如果原文件是链接文件，则复制链接属性
cp - a 相当于 cp -pdr （目标文件与原文件一模一样）

mv [原文件或目录名] [目标目录]

  • mv 为剪切命令

目录简单介绍

文件名 文件内容
/bin 是任意用户都能用到系统命令
/sbin 要超级用户
/boot 启动目录（用户启动数据）
/dev 设备文件保存目录
/etc 配置文件保存目录
/home 普通用户的家目录
/lib 系统库保存目录
/mnt 系统挂载目录
/media 挂载目录
/root 超级用户的家目录
/tmp 临时目录
/sbin 命令保存目录（超级用户才能使用的目录）
/proc 直接写入内存
/usr 系统软件资源目录
/usr/bin/ 系统命令 （普通用户）
/usr/sbin/ 系统命令 （超级用户）
/var 系统相关文档内容

ln 命令

硬链接（非经常用）

ln [原文件] [目标文件]

  • 拥有相同的存储block块和i节点，可以看做是同一个文件
  • 只可通过文件的i节点识别
  • 不能跨分区
  • 不能针对目录使用

软链接（常用）

ln -s [原文件或目录] [目标文件]

  • 类似Windows快捷方式
  • 软链接拥有自己的i节点和block块但是数据块中只保存文件的文件名和i节点，并没有实际的文件数据
  • Irwxrwxrwx I为软链接
    • 软链接文件权限都为rwxrwxrwx
    • 真实权限要看原文件
  • 修改任意文件，另一个都改变
  • 删除原文件，软链接不能使用
  • 原文件需写绝对路径

搜索命令 locate、whereis、find

locate 命令

locate 文件名（后台数据库中只按文件名搜索，搜索树杜快）

  • locate 命令所搜索的后台数据库一天更新一次，所以新创建的文件locate搜索不到
  • updatedb 更新后台数据，不过要首先切换到root用户当中

whereis 命令

whereis 命令名 搜索命令所在路径及帮助文档所在位置，不可查文档

  • whereis -b 只查找可执行命令
  • whereis -m 只查找帮助文件

which 命令

which 命令名 命令所在位置并可以看到他的别名

whereis和which并非每个命令都可以找到，例如cd

find 命令

find [搜索范围] [搜索条件] find是在系统当中搜索符合条件的文件名。如果需要匹配，使用通配符，通配符时完全使用。 使用find命令避免大范围搜索，否则会非常耗费系统资源。

命令 命令内容
find [范围] -name [文件名] 例：find /root -name install.log*，在root文件夹内搜索以install.log开头的文件
find [范围] -iname [文件名] 搜索文件名不区分大小写
find /root -user root 按所有者搜索
find /root -nouser 查找没有所有者的文件，没有所有者的文件可能是内核产生、外来文件、或者是垃圾文件
find /var/log/ -mtime +10 查找十天前修改的文件
-mtime -10 十天内修改的文件
-mtime 10 十天当天修改的文件
-mtime +10 十天前修改的文件
-atime 文件访问时间
-ctime 改变文件属性时间
-mtime 修改文件内容时间
find . -size 25k 查找文件大小是25KB的文件(命令中的k小写)
find . -size 25M 查找文件大小是25MB的文件(命令中的M大写)
-25 小于25的文件
25 等于25的文件
+25 大于25的文件
find . -inum 262422 查找i节点是262422的文件
find /etc -size +20k -a -size -50k 查找/etc/目录下 大于20KB且小于50KB的文件
-a and 逻辑与
-o or 逻辑或
find /etc -size +20k -a -size -50k -exec ls -lh {} \;
查找/etc/目录下 大于20KB并且小于50KB的文件，并显示详细信息
-exec/-ok 命令 {} \; 对搜索结果执行操作 注意空格

grep 命令

grep [选项] [字符串] [文件名] 搜索字符串命令，在文件当中匹配符合条件的字符串，包含匹配，使用正则表达式进行匹配 ，与find的完全匹配不同。

命令 命令内容
greb -i 忽略大小写
greb -v 排除指定字符串
