linux 远程连接

xrdp 模式远程连接(ubuntu12.04,16.04可用)

  • 首先安装 xrdp，tightvncserver

  • 然后在Dash中找到Desktop Sharing，打开并设置

  • 打开本机电脑(win10)找到自带远程桌面控制，输入虚拟机ip

  • 选择sesman-Xvnc模式，然后输入ubuntu账号密码即可

参考网站

https://jingyan.baidu.com/article/5225f26baf6cf3e6fa0908e5.html
（发现可以不用安装dconf-editor）

samba 模式远程连接

  • 首先安装 samba

  • 建立一个要共享文件的目录 例如/home/android/share

  • 给这目录添加权限 （sudo chmod 777 /home/android/share）

修改/etc/samba/smb.conf配置

[share] #该共享的共享名
comment=smb share test
path = /home/android/share # 共享路径
guest ok = yes #其实和public是一样的
available = yes #设置该共享目录是否可用
browseable = yes #设置该共享目录是否可显示
public = yes #指明该共享资源是否能给游客帐号访问
writable = yes #可写

  • 发现配置文件中如果有注释就会报错，所以可以删掉注释

[share]
comment=smb share test
path = /home/android/share
guest ok = yes
available = yes
public = yes
writable = yes

命令重启

sudo service smbd restart

ssh模式连接

  • 首先安装 openssh-server

  • 启动ssh /etc/init.d/ssh start

  • 下载putty

  • 如果出现加载密钥失败 可删除重新生成密钥

    sudo rm -r /etc/ssh/ssh*key
    sudo dpkg-reconfigure openssh-server
